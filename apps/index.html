# KGRID Demo Apps

Cancer Risk Collection

Cancer Risk Collection provides services for the cancer risk score calculation and interpretation.

Script Numerate

Script Numerate Collection can quickly determine whether a prescription has a common, rare, or unprecedented "Sig"

LabWise Collection

LabWise Collection showcases the implementation of test guidelines in a computable form.

Post PCI Collection

Post PCI demonstrates the risk score calculation and visualization. (SMART Sandbox integrated)

CPIC Collection

CPIC Collection focuses on dosing guidelines as provided by the CPIC pharmacogenomic guidelines.

Example Collection

Example Collection demonstrate features of the grid, good development practices and useful tools.
